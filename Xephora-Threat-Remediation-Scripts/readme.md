# Xephora Adware Remediation Script Collection

### Actions

* Terminates processes associated with Adware
* Remove Adware from the file system.
* Removes Adware uninstall key.
* Removes Adware registry keys.
* Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description

These scripts were intended for CrowdStrike but can be run without it.
